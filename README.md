# painel-senhas-node
# 🎬 Sistema de Painel de Senhas com Vídeo

## 📌 Sobre o projeto

Sistema desenvolvido em Node.js que exibe um vídeo em loop e interrompe automaticamente para mostrar senhas recebidas em tempo real via WebSocket.

---

## 🚀 Tecnologias utilizadas

* Node.js
* Socket.IO
* VLC
* TCP Socket
* Automação Windows (nircmd)

---

## ⚙️ Como funciona

1. Inicia o VLC em tela cheia
2. Conecta a um servidor via WebSocket
3. Recebe eventos de senha
4. Pausa o vídeo automaticamente
5. Exibe o painel de senhas
6. Retoma o vídeo do mesmo ponto

---

## 🎯 Diferenciais

* Integração com software externo (VLC)
* Controle de janelas no sistema operacional
* Comunicação em tempo real
* Sistema totalmente automatizado
